## How to Sign

To add your name:

1. Fork this repository
2. Edit `SIGNATURES.md`
3. Add your name in alphabetical order using this format:

```
- Your Name (optional: Company, @github)
```

4. Submit a Pull Request

## Requirements

* You must have read and agreed with the letter
* Your submission must follow the formatting rules
* Keep the list alphabetically sorted
